About the Hunger Games Simulator

The Hunger Games Simulator is an entirely client-side, in-browser implementation of the Hunger Games (by Suzanne Collins), modified to add phrases and scenarios relevant to High Tech life and inside jokes. The code is written entirely in pure HTML, CSS, and Javascript and is hosted on Github.

This specific simulator is actually a fork by Eric Zheng of the original simulator made by Adrian Thamburaj. The original simulation can be found here, and the source is available here. This new simulator is actually the result of Eric redoing the layout of the original simulator, keeping the underlying code written by Adrian.

After this, Adrian released two more versions: (1) an incremental, mostly abandoned version incorporating a few of the stylistic changes that Eric made, and (2) a more complete redesign based on Eric's fork. The latest official release of the simulator can be found here, and the source is available here.
